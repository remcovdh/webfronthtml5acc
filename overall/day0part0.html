Preparation for day 1

As this is our first try-out we (and I myself especially) will probably learn a lot. The prerequisites for this course are a laptop to work on, together with:

  • modern browser: preferably Chrome (https://www.google.com/intl/en/chrome/browser/)
  • node installed, see node.js (http://nodejs.org/)
  • git installed, see (http://git-scm.com/downloads)
  • a working wifi account. Get internet working within Devoteam. If you currently have no account (mail me!!)
  • decide on the editor you like to use and have it working
  • humor (no install url found)
Let me know if there are any problems with installing these tools so we can try to fix them before we start. As the sprint length was previously set on 7 hours including pauses we probably need to set some priorities. But as agile as we try to be and as we are all product owners we can set these priorities and influence this and the next sprints.

For this first evening sprint the following items are currently on the product backlog:

  • Javascript development tools
  • Testing
  • Functional programming
  • Objects in javascript
  • Single page application
  • Event loop and time
  • Advanced a-sync topics
Always with a focus on speeding up the development process for web front-end development and extra possibilities related to html5.

The proposed focus areas of this sprint and next sprints are:

  • this sprint: javascript
  • html 5 features
  • looks, style, devices etc
  • frameworks, architecture
  • to be decided: angular
